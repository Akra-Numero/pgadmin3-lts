Options de pgAdmin - Onglet 5 (Fichiers pour les requêtes)

  • Favourites file path - Chemin vers le fichier contenant les favoris de l'éditeur de requêtes.
     
  • Macros file path - Chemin vers le fichier contenant les macros de l'éditeur de requêtes.
     
  • History file path - Chemin vers le fichier d'historique des requêtes exécutées.
     
  • Maximum queries to store in history - Nombre maximum de requêtes stockées dans le fichier d'historique. Les nouvelles remplacement les plus anciennes.
     
  • Maximum size of a stored query - Toutes les requêtes dont la taille dépasse cette valeur ne seront pas enregistrées dans le fichier d'historique.
