3.6. Conclusion

PostgreSQL has many features not touched upon in this tutorial introduction, which has been oriented toward newer users of SQL. These features are discussed in more detail in both the PostgreSQL 7.3.3 User's Guide and the PostgreSQL 7.3.3 Programmer's Guide.

If you feel you need more introductory material, please visit the PostgreSQL web site for links to more resources.
