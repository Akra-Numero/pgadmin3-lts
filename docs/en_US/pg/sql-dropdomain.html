DROP DOMAIN

Name

DROP DOMAIN -- remove a domain

Synopsis

DROP DOMAIN domainname [, ...]  [ CASCADE | RESTRICT ]

Description

DROP DOMAIN will remove a domain. Only the owner of a domain can remove it.

Parameters

domainname

The name (optionally schema-qualified) of an existing domain.

CASCADE

Automatically drop objects that depend on the domain (such as table columns).

RESTRICT

Refuse to drop the domain if any objects depend on it. This is the default.

Diagnostics

DROP DOMAIN

Message returned if the command was successful.

ERROR: RemoveDomain: type 'domainname' does not exist

This message occurs if the specified domain does not exist.

Examples

To remove the domain box: 

DROP DOMAIN box;

Compatibility

This command conforms to the SQL standard.
