37.4. Global Values in PL/Perl

You can use the %_SHARED to store data between function calls. WHY IS THIS A HASH, AND NOT A HASH REF?

For example: 

CREATE OR REPLACE FUNCTION set_var(TEXT) RETURNS TEXT AS $$
    $_SHARED{first} = 'Hello, PL/Perl!';
    return 'ok';
$$ LANGUAGE plperl;

CREATE OR REPLACE FUNCTION get_var() RETURNS text AS $$
    return $_SHARED{first};
$$ LANGUAGE plperl;

SELECT set_var('hello plperl');
SELECT get_var();
