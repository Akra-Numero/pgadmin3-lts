pgAdmin3 CVS snapshots in source and binary formats

pgAdmin3 CVS snapshots are available for several platforms, including GNU/Linux and Windows. We are doing our best to update these snapshots daily.

Please report bugs, problems and suggestions to the pgadmin mailing list: pgadmin-support@postgresql.org or pgadmin-hackers@postgresql.org. A registration on the mailing-list is needed before posting a message.

There is no guarantee that these packages work in any way: it is beta developer software intended for testing and development.

     
  1. GNU/Linux RedHat 9
    Packager: Jean-Michel POURE

    • Binary RPM packages

      pgAdmin3 binary RPM packages are compiled statickly against wxGTK 2.5. Therefore there is not need to install wxGTK 2.5 packages. pgAdmin3 binaries are compiled with Unicode and debugging mode and linked against GTK2. The binaries are localised in several languages and include the latest PostgreSQL documentation. The required steps for installation are:

      1. Download the latest pgAdmin3 RPM package from http://snake.pgadmin.org/linux/redhat9/RPMS.
      2. Install the downloaded RPM using the following command:
        # rpm -Uvh pgadmin3*.rpm
      3. pgAdmin3 RPM will check for the needed dependencies and warn you if some packages are not installed. All needed dependencies are included in the stock RedHat 9 distribution.

    • Development environment

      If you want to contribute pgAdmin3 projects or simply compile from source, you need a recent GNU/Linux distribution with wxGTK 2.5 libraries, header files and contributed libraries. For your convinience and to avoid long compilation times (sometimes hours), we provide the latest wxGTK 2.5 needed environment in RPM format.

      1. Download the four latest wxGTK2ud RPM packages from http://snake.pgadmin.org/linux/redhat9/wxGTK. The packages are numbered:
          - wxGTK2ud-$version-.i386.rpm
          - wxGTK2ud-devel-$version.i386.rpm
          - wxGTK2ud-xtc-$version.i386.rpm
          - wxGTK2ud-stc-$version.i386.rpm
      2. Install the downloaded RPM using the following command:
        # rpm -Uvh wxGTK2ud*.rpm
      3. wxGTK2ud RPMs will check for the needed dependencies and warn you if some packages are not installed. All needed dependencies are included in the stock RedHat 9 distribution.
