CVS Access

The source code for pgAdmin is available for anyone to use subject to the terms of the Artistic Licence. The pgAdmin III source may be found in the pgadmin3 module. Previous versions of pgAdmin may also be checked out.

CVS access is available through a web interface at cvs.pgadmin.org, or using your favourite CVS client from the repository. Anonymous read access is available at the following CVSRoot (no password is required):

:pserver:anonymous@cvs.pgadmin.org:/disk1/cvsroot
